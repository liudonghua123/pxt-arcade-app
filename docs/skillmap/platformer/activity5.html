AI Rules

The code for this program spawns enemies from the purple [ ! ] tiles. Once the enemies spawn, they immediately start moving to the left and get stuck on a wall‚Ä¶so, let‚Äôs add logic to prevent the enemies from getting stopped.

The enemies will need to follow two rules:

  1. If the enemy is about to run into a wall, it will try to jump over it
  2. If the enemy does hit a wall, it will turn around

Each of these rules has a condition and an action.

If the condition is met, the action will happen. We‚Äôll need to write code to constantly check for each of these conditions.
