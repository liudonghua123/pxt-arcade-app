Introduction

Let’s make a Talent Show clicker game where you decide who is on stage!

Check your scene

  • :binoculars: Look at the game window to see how your scene looks so far.

You should see your character posing in the spotlight on stage.

Step 8

It’s time to add a splash screen to let the user know to press the (A) button when they’re ready to start playing.

~hint What is a “splash screen”? 💡

In video games, a splash screen is a message that shows while a program or level is loading. hint~

  • :circle: Open the ||game:Game|| category and drag
    game.splash("Press (A) to play!")
    into the end of the
    ||loops(noclick):on start|| container that’s already in the workspace.

~hint Click here to see how 🕵🏽

hint~

scene.setBackgroundImage(assets.image`stage`)
let talent = sprites.create(assets.image`towering turtles`, SpriteKind.Player)
talent.bottom = 115
//@highlight
game.splash("Press (A) to play!")

Step 9

⚡ Here comes the action ⚡

The game needs code to make the score go up when the (A) button is clicked.

  • :game: From the ||controller:Controller|| category, grab the
    controller.A.onEvent(ControllerButtonEvent.Pressed, function () {
  info.changeScoreBy(1)
})
    bundle and drop it into an empty area of the workspace.

~hint Click here to see how 🕵🏽

hint~

controller.A.onEvent(ControllerButtonEvent.Pressed, function () {
    info.changeScoreBy(1)
})

Step 10

Play your game on the game screen
🎮 🎮 🎮

Do you see your score go up each time you press the (A) button?

Step 11

This game is off to a great start…but it goes on forever! Add a countdown timer to give the game an ending.

  • :id card: Open the ||info:Info|| category and drag the
    info.startCountdown(10)
    block into the bottom of the
    ||loops(noclick):on start|| container.

~hint Click here to see how 🕵🏽

hint~

🎉 Congrats 🎉

You have started your very own clicker game!

Play it on the game screen and see if you can get more than 40 clicks.
