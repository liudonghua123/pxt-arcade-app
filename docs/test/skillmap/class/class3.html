Step 3

How many are there?

We need to know how many pieces of paper there are. To get this number, we’ll write code to have the computer count them!

  • :redo: From the ||loops:Loops|| category, drag the
    ||loops:count assignments||
    bundle into the end of the
    ||loops(noclick):on start|| container already in the workspace.

  • :mouse pointer: Click on the checkered square, then select the piece of paper from the image grid.

    //@highlight
    info.setScore(stopwatch.getTimerValue(stopwatch.TimerGran.Tenths))
    //@highlight
    game.setGameOverScoringType(game.ScoringType.LowScore)
    //@highlight
    game.gameOver(true)
})
