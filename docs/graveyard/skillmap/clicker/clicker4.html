Introduction

What’s more fun than pressing buttons?

🛒 Buying things 🛒

Let’s modify our clicker game to offer a simple shop.

Step 1

The code for a clicker game is already in the workspace.

When you click Ⓑ, you get a super-clicker — but what fun is it to get extra power for free? Let’s add code to charge the player 10 points for the purchase.

► From the ||info:Info|| category, grab a ||info:change score by [1]|| block and snap it into the top of the on B button pressed container.
